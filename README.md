# Tistory Blog Skin Development Project

이 프로젝트는 티스토리 블로그 스킨을 개발하기 위한 표준 구조와 가이드를 제공합니다. AI 에이전트와 협업하여 현대적이고 성능이 뛰어난 블로그 스킨을 제작할 수 있도록 설계되었습니다.

## 📁 프로젝트 구조

```text
.
├── skin.html          # 블로그의 HTML 뼈대 및 티스토리 치환자 삽입
├── style.css          # 블로그 스타일링 (CSS)
├── index.xml          # 스킨 정보, 변수 및 설정 정의
├── AGENTS.md          # AI 에이전트를 위한 상세 기술 명세서 (치환자 사전 포함)
├── script.js          # (선택) 동적 기능을 위한 자바스크립트
└── images/            # 스킨에 사용될 이미지 및 에셋 폴더
```

## 🚀 시작하기

1.  **개발**: `skin.html`과 `style.css`를 수정하여 디자인을 구현합니다. 티스토리 전용 치환자는 `AGENTS.md`를 참고하세요.
2.  **색상 및 변수**: `index.xml`에 변수를 정의하면 티스토리 관리자 페이지에서 사용자가 직접 디자인을 변경할 수 있습니다.
3.  **테스트**: 
    - 로컬 HTML 파일로 레이아웃을 먼저 잡은 후, 치환자를 적용하는 방식을 권장합니다.
    - 실제 적용은 티스토리 스킨 등록 페이지에서 파일을 업로드하여 확인할 수 있습니다.

## 🤖 AI 에이전트와 협업하기

이 프로젝트에는 AI 에이전트가 스킨의 구조를 완벽하게 이해할 수 있도록 돕는 `AGENTS.md` 파일이 포함되어 있습니다. 새로운 기능을 추가하거나 버그를 수정할 때 이 파일을 AI에게 먼저 읽어보도록 지시하세요.
