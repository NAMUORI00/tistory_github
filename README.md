# 🎨 Tistory Skin — 한반가

티스토리 블로그 스킨 소스 파일입니다.

## 파일 구조

```
├── skin.html        # 스킨 HTML 템플릿
├── style.css        # 스킨 스타일시트
├── script.js        # 스킨 자바스크립트
├── index.xml        # 스킨 설정 메타데이터
├── images/          # 스킨 에셋 이미지
└── preview*.jpg/gif # 스킨 미리보기
```

## 사용법

이 브랜치의 파일을 티스토리 관리 페이지 `꾸미기 > 스킨 편집 > 스킨 등록`에서 업로드합니다.

## 규칙

- 티스토리 표준 치환자(`[##_..._##]`)와 그룹 태그(`<s_...>`)만 사용
- Node.js, npm, 빌드 도구 코드 포함 금지
- 경로 참조 시 `images/파일명` 형식 사용
